# 如何寻找相似用户

1. 曼哈顿距离（Manhattan Distance）

|x1-x2|+|y1-y2|

优点：计算速度快

2. 欧式距离（Euclidean Distance）

√(x1-x2)^2+(y1-y2)^2

3. 明氏距离

两点，P(x1,x2,...,xn), Q(y1,y2,...,yn)∈Rn之间的明可夫斯基距离定义为

(∑|xi-yi|^p)^(1/p) (i=1,2,...n)
其中p=1为曼哈顿距离,p=2为欧氏距离

明氏距离缺点：
主要表现在两个方面：第一，它与各指标的量纲有关；第二，它没有考虑指标之间的相关性。
